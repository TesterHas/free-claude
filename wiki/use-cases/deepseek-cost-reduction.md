# DeepSeek + Claude Code: 100x Cost Reduction

**Source:** Jack Roberts — "DeepSeekV4 + Claude Code = 100X Cheaper" (April 2026, 17 min)

## The Setup

Claude Code framework + DeepSeek V4 backend = same Claude Code UX at a fraction of the cost.

Jack Roberts walked through:
1. A repo that takes minutes to set up
2. No rate limitations (vs Claude's usage limits)
3. Works for complete beginners

## Who this is for

- Developers hitting Claude's rate limits on heavy agentic workflows
- Builders who want to prototype extensively without worrying about API costs
- Clients where the Claude API cost is a blocker in the proposal

## The Trade-offs

See [[../models/deepseek-v4]] for the full picture. Key considerations:
- Data privacy (Chinese lab)
- Performance on complex tasks vs Claude Opus 4.7
- Not suitable for regulated/sensitive workloads

## Sales Strategy

Use DeepSeek + Claude Code as the **proof-of-concept environment**. Build fast, validate the use case, then migrate to Claude API once the value is proven and budget approved.

"We can build this today at essentially zero cost, and if it works, we move to Claude for production."

This removes the budget objection from early-stage conversations.
