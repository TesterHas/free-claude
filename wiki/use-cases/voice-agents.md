# Voice Agents

**Source:** Nate Herk — "Building Realistic Voice Agents Has Never Been Easier" (May 2026, 32 min)

## The Setup

Nate builds a voice agent using **Claude Code + ElevenLabs** — in natural language, no complex infrastructure required. His specific use case:

- He has 400 YouTube videos covering dozens of tools and topics
- Audience gets confused about which tool to use for what
- Solution: voice agent that "chats" with all 400 videos as its knowledge base

The agent lets anyone ask questions and get voice answers synthesized from all his content.

## Why This Matters for Selling

Voice agents are the interface layer that makes AI systems accessible to non-technical buyers. Text chatbots feel like work. Voice feels like talking to someone who knows everything.

Three buyer segments where voice agents win:
1. **Customer support** — 24/7 availability, consistent answers, scales without headcount
2. **Internal knowledge bases** — employees ask instead of search
3. **Content businesses** — turn your content library into a conversational assistant

## Technical stack (Nate's approach)

- Claude Code handles the intelligence layer and orchestration
- ElevenLabs handles voice synthesis
- Build description in natural language → Claude Code builds it
- No framework complexity needed

## Pricing note

Voice agents can command higher margins than text-only systems because:
- Perceived value is higher ("it talks")
- Technical barrier appears higher (even if it isn't)
- Use cases are clearer to non-technical buyers
