# AI Video Editing

**Sources:** Jack Roberts, Nate Herk (April–May 2026)

## The Claim

Both Jack Roberts and Nate Herk covered AI-powered video editing in the same week — "Claude Code just Destroyed Video Editing" (Jack) and "Claude Video Editing Just Became Unrecognizable" (Nate, 28 min).

## What Changed

Claude's video capabilities now include:
- Full editing pipeline from raw footage
- Cuts, transitions, effects from natural language description
- "Unrecognizable" from the previous generation — implying a step change, not incremental improvement

## Jack Roberts' Watch Video Extension

Brad covered "My Claude Code Can INSTANTLY Watch Any Video (Here's How)" — a skill that lets Claude Code watch and analyze any video in seconds. This is the foundation of video-as-input for editing decisions.

## Use Cases for Selling

1. **Content creators** — cut editing time from hours to minutes
2. **Agencies** — scale video production without scaling staff
3. **Course creators** — repurpose long-form recordings into clips automatically
4. **Marketing teams** — generate social media cuts from long-form content

## Stack

- Claude Code as the orchestration layer
- Video input (watch skill) for source analysis
- ElevenLabs or similar for audio (if voice generation needed)
- [[../models/claude-design]] for thumbnail/graphic generation

## Sales note

Video editing is a clear ROI case: a professional editor charges $50–$150/hour. A project that takes 8 hours costs $400–$1,200. If AI does it in 30 minutes, the math sells itself.
