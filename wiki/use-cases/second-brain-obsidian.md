# Claude Code Second Brain in Obsidian

**Source:** Brad Bonanno — "This Claude Code Second Brain Manages My ENTIRE Life in Obsidian (100% Private)" (January 2026, 24 min)

## What it is

Brad's personal operating system: Claude Code integrated directly with Obsidian to manage his life, projects, and knowledge — all locally, no data leaving his machine.

Key distinction: 100% private. Everything runs local. No SaaS, no cloud sync.

## What it does

- Manages tasks, projects, notes within Obsidian
- Claude Code reads and writes to the vault
- Acts as a thought partner on top of the knowledge base
- Pulls together context from multiple notes to answer questions or plan

## Why This Is a Product

This is the base template for an AI-native personal productivity system that can be packaged and sold:
- Obsidian vault structure
- Claude Code CLAUDE.md with context about the person's life/work
- Skills for common workflows (meeting notes, project planning, daily review)
- Hooks for automation

For B2B: replace "personal life" with "company operations" — see Brad's "Company Brain" video (the Slack-reading version).

## The Company Brain Extension

Brad's later video: "I Turned My Second Brain Into a Company Brain (Now it reads Slack for me)" — the same architecture extended to read a company's Slack and feed relevant context into Claude's working memory.

This is the enterprise pitch: a private, secure AI operating system that reads your tools and knows your company.
