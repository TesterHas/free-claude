# Claude Video is Here — Automate Anything (Higgsfield MCP)

**Creator:** Jack Roberts · 17:39 · [Watch](https://www.youtube.com/watch?v=4socWznMV74)
**Farmed:** 2026-05-08

A tutorial on building a content operating system by connecting Claude Code to Higgsfield's MCP for video, image, and animation generation. Shows the full pipeline from website build to animated video component to publish. Includes the "Characters" feature for consistent AI-generated avatars.

## Key Points

- Higgsfield MCP gives Claude Code access to 50+ generation models (Gemini, Nano Banana, Flux, Midjourney, Sora V3, Runway Gen 4, Kling, ElevenLabs)
- CLI integration recommended over MCP standalone — all tools share one connection
- Nano Banana 2 used as Photoshop tool; GPT Image 2 as creator tool
- Characters feature: upload 20+ photos, create character, reference by name in Claude prompts
- Key workflow: Claude Code builds website → Higgsfield generates images → Higgsfield animates (video gen) → Publish via Vercel/GitHub CLI
- Every workflow can be saved as a reusable skill.md
