# How I'd Learn Claude From Scratch in 2026

**Creator:** Brad | AI & Automation · 11:29 · [Watch](https://www.youtube.com/watch?v=9sp5PCKbvuc)
**Farmed:** 2026-05-08

A step-by-step syllabus for learning Claude in 2026, organized as 13 rungs in order. Brad's thesis: most people use ~2% of what they pay for. The list progresses from basic chat to autonomous routines.

## The 13-Rung Syllabus

1. **Claude Chat** — claude.ai, artifacts (diagrams, apps, slides, Word, Excel in chat)
2. **Connectors** — MCP-based tool wiring (Gmail, Drive, Notion, Calendar, Slack)
3. **Projects** — Self-contained environments with own instructions, knowledge, chat history
4. **Claude Desktop** — Gateway to advanced products (Co-work, Code, Routines)
5. **Co-work** — Sees every file in a folder; opens, edits, creates, runs scripts; non-coders get agent power
6. **Live Artifacts** — Static artifacts that pull fresh data through connectors on refresh; persistent dashboards
7. **Skills** — skill.md folders encoding domain knowledge; work everywhere (chat, co-work, Office)
8. **Dispatch** — Phone-to-desktop task handoff; Claude works at home, result back on phone
9. **Word/PowerPoint/Excel Extensions** — Claude inside Office apps; reads live files, formats, models
10. **Claude in Chrome** — Browser extension; reads screens, clicks, fills forms, records workflows
11. **Claude Design** — Visual workspace: slides, infographics, wireframes, full front-ends
12. **Claude Code** — 5x co-work context window; terminal/VS Code; go-to-market workflows; runs entire content operation
13. **Routines** — Scheduled/event-driven automations (e.g. Fireflies transcript → action items → Slack)

## Key Insight

"The real unlock isn't scheduled routines — it's event-driven ones." Routines that fire on webhooks (Fireflies, Slack events) create permanent leverage. Brad claims 10 routines = "an entire workday back every week forever."
